# Customer Churn Prediction using Machine Learning

## Overview

This project predicts whether a telecom customer is likely to leave the company based on demographic and service-related information.

## Objectives

- Analyze customer behavior and churn patterns
- Preprocess and clean customer data
- Train machine learning classification models
- Compare model performance using evaluation metrics

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics

- Accuracy Score
- Precision
- Recall
- F1 Score
- ROC-AUC Score
- Confusion Matrix

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Train-Test Split
6. Model Training
7. Performance Evaluation
8. Feature Importance Analysis

## Results

The models were evaluated using ROC-AUC, confusion matrix, and classification metrics to identify factors contributing to customer churn.
